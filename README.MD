# HandwrittenImageMiya - 簡単画像注釈ツール

## 概要
画像（PNG, JPG等）やPDFファイルを読み込み、文字や図形などの注釈をサクサク書き込めるデスクトップアプリです。
お仕事の資料作成や、写真へのメモ書きに最適です。

## 主な機能
- **選べる2つのテーマ（新機能!）:** 目に優しい「ダークモード」と、清潔感のある「通常モード（ライト）」をボタン一つで切り替えられます。
- **はっきり見えるツール選択:** どのツールを使っているか一目で分かるよう、ツールバーの表示を改良しました。
- **自由自在な色設定:** 「文字の色」「図形の枠の色」「塗りつぶしの色」を個別に設定できます。
- **直感的なテキスト配置:** 文字を入力した後、マウスを動かして好きな場所にカチッと置くだけ。
- **PDF対応:** PDFファイルを開いて、ページをめくりながら注釈を入れ、そのまま画像やPDFとして保存できます。
- **やり直し機能:** 「戻る」ボタンや Ctrl+Z で、間違えた操作を簡単に取り消せます。

## 動作イメージ
- **🌙 ダークモード:** 夜間や暗い部屋での作業に最適。
- **☀️ 通常モード:** 明るいオフィスや昼間の作業に。
- **📏 多彩なツール:** 直線、矢印、フリーハンド、四角、丸、三角、テキスト入力など。

## インストール・実行方法
1. 配布された `HandwrittenImageMiya-v1.3.0.exe` をフォルダから探します。
2. `.exe` ファイルをダブルクリックして起動します。

### ⚠️ 初回起動時に警告が表示される場合
Windowsの保護機能（SmartScreen）により、「WindowsによってPCが保護されました」という青い画面が表示されることがあります。これは未登録のアプリに対して表示される標準的な警告です。以下の手順で安全に実行できます：
1. メッセージ内の **「詳細情報」** をクリックします。
2. 右下に表示される **「実行」** ボタンをクリックすると、アプリが起動します。
※ 2回目以降の起動では表示されなくなります。

## 基本的な使い方
1. **ファイルを開く:** 「📂 開く」から画像やPDFを選びます。
2. **注釈を描く:** ツールバーからツールを選び、画面上をドラッグして描きます。
3. **色や太さを変える:** 描く前にツールバーで色や太さを指定できます。選択ツールで選んだ後から変更することも可能です。
4. **保存する:** 「💾 保存」ボタンを押して、名前をつけて保存します。
5. **テーマを変える:** 右上の 🌙/☀️ アイコンを押すと、いつでもデザインを切り替えられます。
