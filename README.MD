# README.md

## 概要
画像（PNG, JPG等）やPDFファイルを読み込み、注釈を書き込めるデスクトップアプリケーションです。

## 通常のペイントアプリ以上の特徴
- **保存機能の最適化（NEW!）:** 保存時の初期形式を**JPEG**に設定しました。また、保存ファイル名のデフォルトを自動的に**「[元のファイル名]_After.jpg」**とするよう改良し、手動入力の手間を減らしました。
- **自動で閉じる保存完了通知:** 保存が完了した際の「保存しました」というメッセージは、**3秒後に自動的に閉じます**。
- **残像・表示バグの修正:** 文字や図形の移動時、およびリサイズ時に発生していた青いハンドルの残像（ゴミ）が出ないよう、描画更新処理を最適化しました。
- **ファイル展開時の初期ディレクトリ設定:** 「ファイルを開く」をクリックした際、毎回必ずホームディレクトリからダイアログが開きます。
- **未保存の変更検知機能:** 編集中にファイルを切り替えたり、アプリを終了しようとした際、未保存であれば確認ダイアログが表示されます。
- **リストダウンによる直感的な太さ選択:** 太さの指定UIを、使いやすいドロップダウンリストに変更しました。手入力での直接指定も可能です。
- **矢印の形状変更とバランス調整:** 矢印の先端を塗りつぶしの三角形から、視認性の高い「くの字（V字）」形状に変更しました。
- **起動時の最大化表示:** アプリ実行時、自動的にウィンドウが最大化された状態で起動します。
- **テキストの再編集機能:** 配置したテキストオブジェクトを、右クリックメニューから後から編集できます。
- **Exif回転情報の自動補正:** スマートフォンなどで撮影された写真を開く際、自動的に本来の向きに補正します。
- **PDF対応:** PDFを直接読み込み、ページを切り替えながら注釈を加え、画像として保存できます。

## 必要条件
- Python 3.8 以上
