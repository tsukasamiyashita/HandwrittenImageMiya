# README.md

## 概要
画像（PNG, JPG等）やPDFファイルを読み込み、注釈を書き込めるデスクトップアプリケーションです。

## 通常のペイントアプリ以上の特徴
- **文字色・枠線色・背景色の完全個別設定（NEW!）:** 「文字自体の色」「枠線の色」「背景の塗りつぶし色」の3つをすべて個別に設定できるようになりました。
- **プレビュー付きの直感的なテキスト配置:** テキストツールを選択すると先に入力ダイアログが開き、入力後キャンバス上でマウスにテキストが追従します。視覚的に場所を確認しながらクリックで配置でき、完了後は自動で選択ツールに戻るため非常にスムーズに操作できます。（※右クリックで配置キャンセルも可能）
- **ツールバーの2段構成:** ボタン数が増加したため、ツールバーを2段に分割しました。画面幅が狭い場合でも「ページめくり」や「ズーム」などの操作ボタンが隠れることなく利用できます。
- **前面に配置機能の追加:** 右クリックメニューから、隠れてしまった図形や文字を重ね順の「一番手前」に移動できるようになりました。Undo/Redoにも完全対応しています。
- **Undo/Redo機能:** ツールバーに「戻る（Ctrl+Z）」と「進む（Ctrl+Y）」ボタンを配置。間違えて描画・削除してしまった際も簡単に取り消せます。
- **コンパクトなUI設計:** 作業領域を広くとれるよう、ツールバーのボタンやコンボボックスのデザインを省スペースに調整しました。
- **削除機能:** ツールバーの「削除」ボタン、または「Deleteキー」で選択アイテムを削除できます。
- **保存機能の最適化:** 保存時の初期形式を**JPEG**に設定しました。また、保存ファイル名のデフォルトを自動的に**「[元のファイル名]_After.jpg」**とするよう改良し、手動入力の手間を減らしました。
- **自動で閉じる保存完了通知:** 保存が完了した際の「保存しました」というメッセージは、**3秒後に自動的に閉じます**。
- **残像・表示バグの修正:** 文字や図形の移動時、およびリサイズ時に発生していた青いハンドルの残像（ゴミ）が出ないよう、描画更新処理を最適化しました。
- **ファイル展開時の初期ディレクトリ設定:** 「ファイルを開く」をクリックした際、毎回必ずホームディレクトリからダイアログが開きます。
- **未保存の変更検知機能:** 編集中にファイルを切り替えたり、アプリを終了しようとした際、未保存であれば確認ダイアログが表示されます。
- **リストダウンによる直感的な太さ選択:** 太さの指定UIを、使いやすいドロップダウンリストに変更しました。手入力での直接指定も可能です。
- **矢印の形状変更とバランス調整:** 矢印の先端を塗りつぶしの三角形から、視認性の高い「くの字（V字）」形状に変更しました。
- **起動時の最大化表示:** アプリ実行時、自動的にウィンドウが最大化された状態で起動します。
- **テキストの再編集機能:** 配置したテキストオブジェクトを、右クリックメニューから後から編集できます。
- **Exif回転情報の自動補正:** スマートフォンなどで撮影された写真を開く際、自動的に本来の向きに補正します。
- **PDF対応:** PDFを直接読み込み、ページを切り替えながら注釈を加え、画像として保存できます。

## 必要条件
- Python 3.8 以上
